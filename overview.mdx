## Welcome to Aleesha Kost

This documentation helps you use the **MCP Agent AI Aleesha Kost** to search for room information, view prices and availability, and make direct bookings via the MCP-enabled AI agent.

With this integration, you can interact using everyday language without having to manually open and check each page.

<Info>
  The Aleesha Kost MCP AI Agent is designed to help prospective tenants obtain
  room information and make bookings more quickly, easily, and systematically.
</Info>

## What can be done?

<CardGroup cols={2}>
  <Card title="Looking for a room" icon="Search">
    Find Aleesha Kost rooms based on type, price, amenities, or other requirements.
  </Card>

<Card title="Check availability" icon="Bed">
  Check room stock and availability status based on the latest data from Aleesha
  Kost.
</Card>

<Card title="View room information" icon="Info">
  Get information regarding prices, amenities, terms and conditions, and details
  for each room.
</Card>

  <Card title="Booking a room" icon="CalendarDays">
    Book a room directly through a conversation with an AI Agent.
  </Card>
</CardGroup>

## Example of usage

Once the MCP Aleesha Kost is connected to the AI ​​agent, you can issue commands such as:

- “Show available rooms.”
- “How much does a room cost for one month?”
- “Look for rooms with an en suite bathroom.”
- “Show available amenities.”
- “I want to book a room that is still available.”
- “Please review my booking details.”

The AI ​​agent will forward the request to the Aleesha Kost service and display the relevant results.

## How it works

<Steps>
  <Step title="Connect MCP Aleesha Kost">
    Add and configure MCP Aleesha Kost to the application or AI agent that you
    use.
  </Step>
  <Step title="Submit a request">

    making a reservation.
  </Step>
  <Step title="Agent AI memproses permintaan">
    The AI ​​agent will use the appropriate MCP features to retrieve the latest
    information from the Aleesha Kost system.
  </Step>
  <Step title="Review and confirm">
    Check the provided information before confirming the room reservation.
